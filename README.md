# Neural Network
## Overview
This program was my way of understanding how multilayer perceptrons work. I relied heavily on 3Blue1Brown's YouTube series for the maths and logic behind the machine learning algorithm.

It performs optical character recognition on the digits 0-9 and has been trained on the MNIST database.
